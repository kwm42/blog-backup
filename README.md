# README

一个使用 Vue.js 3 编写的博客程序，备份一下。

### 怎么运行

- 进入目录

```shell
cd blog-backup
```

- 安装依赖

```shell
npm install
```

- 运行

```
npm run serve
```



## How to start

- Enter the directory 

```shell
cd blog-backup
```
